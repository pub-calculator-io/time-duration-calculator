Use our time duration calculator to determine how much time has passed between two events. It is perfect for tracking work time for your business.

[https://www.calculator.io/time-duration-calculator/](https://www.calculator.io/time-duration-calculator/)

== Usage ==

Add the Time Duration Calculator shortcode to your page, post or sidebar:

`[ci_time_duration_calculator]`

Add the following code to your template where you would like the Time Duration Calculator to appear:

`<?php display_ci_time_duration_calculator(); ?>`

== Installation ==

1. Upload the Time Duration Calculator /ci_time_duration_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Time Duration Calculator plugin through the "Plugins" menu in WordPress.
